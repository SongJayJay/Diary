Doxygen
================================================================================================================
## Doxygen简介
   doxygen注释块就是在C/C++注释块的基础上添加的一些额外的标识，使得doxygen把它识别出来，并将它组织到生成的文档中。
## Doxygen注释
   doxygen支持的注释风格：
   1. javaDoc风格 
      ```
      /**
       *......
       *......
       *......
       */
       ```
   2. QT风格
      ```
      /*!
      ....
      ....
      */
      ```
   3. C/C++行注释风格
      //或//!
## Doxygen关键字
   在doxygen文档中有很多@前缀的语句。
   @file 文件名字
   @date 文件生成日期
   @oaram 参数
## Doxygen插件
  在vim环境中为了简化doxygen注释的编写，可以使用doxygenTooKit.vim来实现
  注释的自动化加入。
